INTRODUCTION
------------

Hangman game

 * To start the game go to: hangman

 * By default the word to start playing with is: "hangman".

 * To add new words use: hangman/words

 * You have to be loggin to start using the game.

 * You have 5 trials only.

 * Score is calculated based on the number of trials you used multiply 10.
